1) Получение текущих кусов:
url: http://localhost:9094/sbt-services/services/rest/rateService/rate/current
параметры: 
currencyCode - код валюты, одно из значений, может передаваться массив:
	978 - Евро,
	840 - Доллар,
	A33 - Палладий,
	A76 - Платина,
	A98 - Золото.
rateCategory - категория курса, одно из значений, может передаваться массив:
    base - базовый курс оделения,
	beznal - курс безналичной конверсии,
	premium - курс Премьер,
	first - курс Сбербанк 1.
regionId - идентификатор региона.
date - дата курса в формате dd.MM.yyyy HH:mm:ss. Если параметр пустой, или отсутствует, то возвращается курс на текущую дату.

2) Получение доступных инервалов валют для конверсии валют:
url: http://localhost:9094/sbt-services/services/rest/rateService/rate/ranges
парамеры:
currencyCode - код валюты, одно из значений, может передаваться массив:
	978 - Евро,
	840 - Доллар,
	A33 - Палладий,
	A76 - Платина,
	A98 - Золото.
regionId - идентификатор региона.
sourceCode - код источника "от куда", одно из значений:
	card - карта,
	account - счет,
	cash - наличные.
destinationCode - код источника "куда", одно из значений:
	card - карта,
	account - счет,
	cash - наличные.
exchangeType - метод перевода, одно из значений:
	ibank - интернет банк,
	office - отделение,
	atm - банкомат.
servicePack - пакет обслуживания, одно из значений:
	empty - нет пакета
	premier - премьер
	first - первый.
	
3) Получение истории курсов валют:
url: http://localhost:9094/sbt-services/services/rest/rateService/rate
URL парамеры:
regionId - идентификатор региона.
fromDate - начальная дата в формате dd.mm.yyyy
toDate - конечная дата в формате dd.mm.yyyy

Парметры в теле запроса:
currencyCode - код валюты, одно из значений:
	978 - Евро,
	840 - Доллар,
	A33 - Палладий,
	A76 - Платина,
	A98 - Золото.
rangeId - идентификатор диапазона, полученный в запросе 2, может передаваться массив диапазонов для одной валюты.

Пример тела запроса:
{
  "currencyData": [{
    "currencyCode":"840",
    "rangesAmountFrom":[0, 1500, 3000]
  }]
}

5) Конвертация валют
url: http://localhost:9094/sbt-services/services/rest/rateService/rate/conversion
параметры:
regionId - идентификатор региона.
sourceCode - код источника "от куда", одно из значений:
	card - карта,
	account - счет,
	cash - наличные.
destinationCode - код источника "куда", одно из значений:
	card - карта,
	account - счет,
	cash - наличные.
exchangeType - метод перевода, одно из значений:
	ibank - интернет банк,
	office - отделение,
	atm - банкомат.
servicePack - пакет обслуживания, одно из значений:
	empty - нет пакета
	premier - премьер
	first - первый.
fromCurrencyCode - код валюты, из которой происходит конвертация, параметр не передается, если конвертация из рублей, или передается пустым
toCurrencyCode - код валюты, в которую происходит конвертация, параметр не передается, если конвертация в рубли, или передается пустым
date - дата курса в формате dd.mm.yyyy hh24:mm:ss, если параметр пустой или отсутствует, то конвертация осуществляется по курсу на текущую дату
amount - количество конвертируемой валюты

6) Справочник валют
url: http://localhost:9094/sbt-services/services/rest/rateService/rate/currency
параметры:
type - тип валюты
	METAL - металлы,
	CURRENCY - валюта.
